## FUNCIONALIDAD DE LA PRACTICA

1. Iniciar pin de led como salida 
2. bucle infinito 
    * encender led  
    * espera de 500 milisegundos
    * apagar led 
    * espera de 500 milisegundos

## Código básico

```c
#define LED_BUILTIN 2
#define DELAY 500

void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH);
  delay(DELAY);
  digitalWrite(LED_BUILTIN, LOW);
  delay(DELAY);
}

```
